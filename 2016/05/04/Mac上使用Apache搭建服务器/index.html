Mac上使用Apache搭建服务器

本来是要搞HLS(苹果的视频直播处理技术)的，而且已经将mp4文件分割成m3u8类型直播文件，最后要验证m3u8文件的时候，却发现是要利用服务器来处理的，所以只好查了些服务器的知识。因为Mac自带Apache，所以为了省事就直接学习了下就拿来用了。

启用Apache

  1. 开启apache命令：sudo apachectl start
  2. 关闭apache命令：sudo apachectl stop
  3. 重启apache命令：sudo apachectl restart
  4. 开启apache后，可以直接在浏览器中访问 localhost, 如果出现“It works”，表示运行正常。

具体使用：

目标：在浏览器中打开localhost，显示自定义的web页面。

  1. 在Mac的Finder中，利用前往文件夹，进入路径/Library/WebServer/Documents，这是默认的根路径，里面有三个文件：index.html.en, PoweredByMacOSX.gif, PoweredByMacOSXLarge.gif。这三个文件不用管，放在那里就行。

  2. 利用Coda，或者eclipse创建一个html文件，内容自己定义就行。然后将这个html文件名字改为index.html, 只能是这个名字，除非你利用vim更改Apache的httpd.conf文件，重新配置首页文件名。

  3. 将创建的index.html文件保存到 /Library/WebServer/Documents下。因为这是个私有文件夹，是需要root权限的，所有每次更改这个文件夹都是需要输入Mac密码的，除非Mac没有设定密码。

  4. 最后，浏览器中打开localhost,就可以看到自定义的html。
